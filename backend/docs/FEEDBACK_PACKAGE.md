# Feedback Package

The `FeedbackPackage` is the strictly typed and standardized JSON object generated by the `FeedbackBuilder`.
It acts as the single source of truth for the Learning Engine (Sprint 9.9) to train and adjust the prompt strategy.

It normalizes all AI evaluation, decision logic, user interaction, and physical real-world performance metrics into unified scoring structures: `feedback_score`, `edit_score`, `performance_score`, and `overall_score`.
